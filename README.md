# Data-Analytics

## 📊 Análisis de Venta y Perfiles del cliente en EEUU🚘


Descripción

En este proyecto se realizará un EDA y una segmentación en un historial de ventas en distintas cadenas dentro del sector del automóvil dentro de distintas regiones de los EEUU. El objetivo será identificar el perfil de cada comprador dentro de cada región y ver que equipamiento (marca, tipo de vehículo, motor...) es el que más beneficio otorga.
